# Example Emojis

This folder contains your custom emoji images.

## Supported Formats
- PNG (recommended for transparency)
- GIF (for animations)
- JPG/JPEG
- WEBP
- SVG

## Naming Convention
Use descriptive, lowercase names with hyphens:
- ✅ `happy-cat.png`
- ✅ `thumbs-up.gif`
- ✅ `party-parrot.png`
- ❌ `Image1.PNG`
- ❌ `My Emoji.jpg`

## Adding Emojis
Simply add your image files to this folder and commit them to the repository.
